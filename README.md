# Cljs::Rails

Boot powered helpers to get up and running fast with Clojurescript in your Rails application

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'cljs-rails'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cljs-rails
